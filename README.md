# AI Research Team

A user can enter the topic and get all the details regarding the topics with MCQ.

## Overview

This project provides a comprehensive learning tool where users can explore various topics and test their knowledge through multiple-choice questions (MCQs).

## Features

- **Topic Search**: Enter any topic to get detailed information
- **Comprehensive Details**: Access in-depth content about selected topics
- **MCQ Assessment**: Test your knowledge with multiple-choice questions
- **Research-Driven**: Built for research and learning purposes

## Technology Stack

- **Language**: Python 100%

## Getting Started

### Prerequisites

- Python 3.x installed on your system
- Required dependencies (see requirements.txt)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/25pai028-art/ai_research_team.git
cd ai_research_team
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Set up environment variables (if needed):
```bash
cp .env.example .env
# Edit .env with your configuration
```

### Usage

Run the application:
```bash
python main.py
```

Then follow the on-screen prompts to:
1. Enter a topic of interest
2. Review detailed information
3. Complete MCQ assessments
